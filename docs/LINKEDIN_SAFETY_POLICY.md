# LinkedIn Safety Policy

This extension is made to reduce manual sales work. It is not made to automate LinkedIn behavior.

## What The Extension Does Not Do

- It does not automatically browse LinkedIn.
- It does not automatically send DMs.
- It does not read cookies.
- It does not read localStorage or sessionStorage.
- It does not scrape many profiles.
- It does not extract hidden information.

## What The Extension Does

- It works only on the currently opened LinkedIn profile page.
- It uses only information that is visible on the page.
- It runs profile analysis only after the user clicks a button.
- It drafts messages that the user must review and send manually.

## User Responsibility

Users should review LinkedIn's terms before using this extension.

Users should avoid high-frequency or bulk usage. The safest use is slow, manual, and focused on one profile at a time.

Users are responsible for deciding whether a generated message is accurate, respectful, and appropriate.
