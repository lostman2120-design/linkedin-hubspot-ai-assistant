# v0.4.0 Release Checklist

## Theme

Lead Decision + HubSpot Sales Context

The extension should help a user decide whether a visible LinkedIn lead is worth pursuing and save the reason, risks, outreach strategy, and next action to HubSpot.

## HubSpot Scopes

The Private App needs the existing contact, note, and task scopes. Automatic LHA property setup also needs contact property schema read/write scopes. If schema access is missing, standard contact and note saving continue.

## Known Limitations

- LinkedIn DOM extraction depends on information visible on the currently open profile.
- Outreach Strategy is a cautious AI hypothesis, not confirmed buyer intent.
- HubSpot admins may need to grant property schema access before LHA properties can be created.
- A Free or limited HubSpot account may restrict some task or property APIs.
- Real Chrome, LinkedIn, OpenAI, and HubSpot integration checks remain manual.
