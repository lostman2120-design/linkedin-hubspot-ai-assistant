# v0.5.0 Release Checklist

Theme: Sales Decision Intelligence.

Main message: Improve the quality of the sales decision, not just the speed of outreach.

## Automated Checks

- [ ] `corepack pnpm lint`
- [ ] `corepack pnpm test`
- [ ] `corepack pnpm build`
- [ ] Confirm `apps/extension/dist/manifest.json` says `0.5.0`.
- [ ] Confirm the upload zip does not include `.env`, secrets, `node_modules`, source maps, or test files.

## Sales Decision QA

- [ ] Recommended Action appears at the top.
- [ ] Outreach Readiness appears below Recommended Action.
- [ ] ICP Fit Score and Fit Label are visible.
- [ ] Decision Confidence, Data Sufficiency, and Evidence Coverage are visible.
- [ ] Decision Breakdown is collapsible and includes all nine fields.
- [ ] What would change this decision? appears with conditional language.
- [ ] Next Best Research Action uses only safe manual research suggestions.
- [ ] AI Outreach Coach appears before DM Drafts and always says human review is required.

## Regression Fixtures

- [ ] Bill Gates-style public/nonprofit profile is not Strong fit.
- [ ] Joris-style limited HubSpot/RevOps consultant profile scores about 65-82.
- [ ] Rich HubSpot agency profile reaches 85-95 only when independent evidence is visible.
- [ ] Founder-only profile stays 54 or lower.
- [ ] LinkedIn URL text is not counted as pain evidence.

## HubSpot QA

- [ ] Add to HubSpot still creates or updates the correct contact.
- [ ] No contact is created with `Unknown`, `N/A`, or `Unable to extract this field`.
- [ ] HubSpot note metadata says `LinkedIn to HubSpot AI Assistant v0.5.0`.
- [ ] HubSpot note includes Decision Breakdown, Outreach Readiness, Next Research Actions, and AI Outreach Coach.
- [ ] Optional LHA custom property failures show a warning but do not block contact, note, or task creation.

## Safety QA

- [ ] No automatic LinkedIn browsing.
- [ ] No automatic LinkedIn DM sending.
- [ ] No hidden LinkedIn data extraction.
- [ ] No cookie, localStorage, or sessionStorage access.
- [ ] Users must still click buttons before analysis, DM generation, or HubSpot writes.
